# Changelog

## [0.2.0] - 2026-03-22

### Added

- **文字種境界スコアリング**: 漢字↔かなの境界位置を検出し、自然な分割位置にボーナス、不自然な位置にペナルティを適用
- **OOV 姓の文字種混在ペナルティ**: 辞書未収録の姓にひらがな/カタカナが混じっている場合に減点（日本人の苗字はほぼ漢字のみ）
- **姓ヒットボーナス** (+0.5): 姓辞書ヒットを名辞書ヒットより優先（苗字は有限の既知セット）
- **カタカナ姓の例外フロー**: 前半が全カタカナの場合、後半を姓辞書で照合して芸名パターンとして判定（例: ジャガー/横田）
- **かな→漢字方向の境界スコアリング**: 漢字→かなだけでなく、かな→漢字の境界も検出・スコアリング
- **テストデータ拡充**: 12カテゴリ 339件（MVP 208件 + エッジケース 131件）
  - VTuber名、漢字+ひらがな/カタカナ、かな+漢字、1文字姓、3文字以上姓、異体字、ノ・ヶ・々、全かな、曖昧分割
- **allowLowConfidence モードのテスト**: 各カテゴリで通常/低信頼の両モードの精度を計測
- **CI テストワークフロー**: PR 時に自動テスト + カテゴリ別精度レポートをコメント投稿
- **CI docs 自動更新**: main push 時に docs/seimei-split.js を自動再ビルド
- **ドキュメントサイト**: TanStack Router + React + Tailwind CSS で SPA を構築
  - Playground（textarea 複数行入力、サンプルカテゴリ、エクスポート機能）
  - Docs（API リファレンス）
  - How it works（アルゴリズム解説）

### Changed

- README.md を日本語化、英語版は README.en.md に移動
