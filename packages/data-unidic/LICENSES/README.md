## Data Source

This package contains dictionary data derived from **UniDic** (現代書き言葉UniDic).

- **Source**: National Institute for Japanese Language and Linguistics (NINJAL / 国立国語研究所)
- **Version**: 202512
- **URL**: https://clrd.ninjal.ac.jp/unidic/
- **Extraction method**: Filtered `lex.csv` for entries with POS `名詞,固有名詞,人名,{姓|名}`
