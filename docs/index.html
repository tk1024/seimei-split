使い方

インストール

npm install seimei-split

基本的な使い方

辞書が同梱されているので、インポートするだけで使えます。

import { split } from "seimei-split";

split("田中太郎");    // => { sei: "田中", mei: "太郎" }
split("田中 太郎");   // => { sei: "田中", mei: "太郎" }
split("さとうはなこ"); // => { sei: "さとう", mei: "はなこ" }

詳細な分析

analyze() を使うと、信頼度スコアや候補一覧を取得できます。

import { analyze } from "seimei-split";

const result = analyze("田中太郎");
// result.best       => { sei: "田中", mei: "太郎" }
// result.confidence  => スコア値
// result.candidates  => 全候補の配列

オプション

信頼度が低い場合でもベストエフォートで結果を返すには allowLowConfidence を指定します。

split("珍しい名前", { allowLowConfidence: true });

辞書なしで使う（軽量版）

seimei-split/core からインポートすると辞書を同梱せずに使えます（約 3.5 KB）。setLexicon() で独自の辞書を設定してください。

import { split, setLexicon } from "seimei-split/core";

setLexicon(myCustomLexicon);
split("田中太郎");
